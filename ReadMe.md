# 博客源码详细说明
* 写博客过程参考已经发表的文章
* 在Blogs目录下相应的时间范围文件夹内把博客html文件放进去（添加完整博客）
* 在archives目录下（添加博客名称、类别、标签--依据年限）
* 在categories目录下对博客进行分类（添加博客名称、类别、标签--依据类别）
* 在page目录下首页加载，即全部文章（包括摘要）
* 在tags目录下相应标签放相应文章
* 修改search.xml文件（搜索用的）
